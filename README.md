# Cozy Reader ☁️

A personal audiobook player with a Girly Hand-Drawn Japanese INS Style.

## Features
- **Visuals**: Cream paper texture, Washi tape decorations, Polaroid-style book covers.
- **Tech Stack**: React, Vite, Tailwind CSS (via clsx/tailwind-merge), Framer Motion.

## How to Run
1. Open terminal in this directory.
2. Run `npm install` (if you haven't already).
3. Run `npm run dev`.
4. Open the link (usually `http://localhost:5173`) in your browser.

## Project Structure
- `src/App.jsx`: Main application logic and components (Bookshelf, Player).
- `src/index.css`: Global styles, fonts, and CSS variables for the hand-drawn look.
